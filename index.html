Mis Proyectos

Tengo algunos proyectos que demuestran mis habilidades en desarrollo web. Todos han sido realizados durante el curso de programación web que dicta academia LEXPIN. Cree una plataforma similar en apariencia a Youtube utilizando solo HTML Y CSS, además replique la página del perfil de Lexpin en instagram utilizando HTML, CSS Y SASS. Finalmente, realice un proyecto utilizando WordPress, lo que ha hecho crecer mis hablidades en la personalización CMS y la entrega rápida de soluciones web funcionales. Estos proyectos me han permitido dominar diversas tecnologías y metodologías de desarrollo, demostrando mi capacidad para enfrentar y resolver desafios técnicos.

Youtube

Clon visual de la interfaz de YouTube. Este proyecto se centra en la maquetación front-end, replicando la estructura y el diseño visual de la página principal de YouTube. Fue desarrollado utilizando exclusivamente HTML5 y CSS3 para demostrar habilidades en diseño responsive y estructuración semántica sin implementar funcionalidad interactiva real. (2025)

Lexpin

Replica del perfil de Instagram de Lexpin. Un ejercicio práctico de diseño web enfocado en la clonación de la interfaz de usuario de una página de perfil específica de Instagram. El objetivo fue practicar la implementación de un diseño complejo utilizando HTML, CSS y SASS, haciendo uso de variables y anidamiento para un código más eficiente y mantenible. (2025)

Calculadora

Este proyecto es una demostración integral de la versatilidad de JavaScript, presentando tres calculadoras básicas desarrolladas con metodologías distintas: Programación Funcional, Programación Orientada a Objetos (POO) y Manipulación del DOM basada en eventos. La aplicación principal sirve como índice que enlaza a estas implementaciones, permitiendo una comparación directa de cómo cada paradigma gestiona la lógica matemática y la interfaz de usuario, utilizando HTML5, Tailwind CSS y JavaScript. (2025)
