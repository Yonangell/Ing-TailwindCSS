Mis Proyectos

A lo largo de mi proceso de formación como desarrollador web he trabajado en distintos proyectos enfocados en la creación de sitios web funcionales, responsivos y orientados a la experiencia del usuario.

Mis proyectos incluyen desarrollos realizados tanto con tecnologías frontend (HTML, CSS, JavaScript y Tailwind CSS) como con WordPress, donde he trabajado en la personalización de temas, estructura de contenidos y adaptación del diseño según las necesidades del proyecto.

En cada proyecto me enfoco en:

  • Construir interfaces claras y usables
  • Aplicar diseños modernos y responsivos
  • Mantener una estructura ordenada y comprensible
  • Adaptar soluciones según la tecnología utilizada

Estos proyectos reflejan mi capacidad para trabajar con diferentes herramientas, entender el flujo completo de un sitio web y seguir aprendiendo continuamente para mejorar mis habilidades como desarrollador.

Jankenpon

Implementación de una plataforma interactiva del clásico juego 'Piedra, Papel o Tijera', diseñada bajo un enfoque de código limpio y alto rendimiento mediante el uso de JavaScript (ES6+) y Tailwind CSS. Administrador de usuario

Desarrollo de un Administrador de Usuarios robusto, diseñado como una aplicación tipo CRUD que permite la gestión dinámica de registros en tiempo real. El proyecto está construido con JavaScript (ES6+) y Tailwind CSS, integrando una lógica avanzada para la validación de formularios y la persistencia de datos mediante el uso de LocalStorage, lo que garantiza que la información se mantenga tras la recarga de la página. (2025)

ING-SYSTEM

ING-SYSTEM se consolida como una plataforma de e-commerce de alto rendimiento dedicada a la curaduría de moda para damas, caballeros y accesorios de diseño. La arquitectura del sitio ha sido desarrollada bajo estándares modernos de diseño web, utilizando el ecosistema de WordPress en combinación con el tema Astra, lo que garantiza una estructura ligera, veloz y altamente optimizada para SEO. El despliegue visual y la experiencia de usuario (UX) han sido modelados mediante Elementor, permitiendo una interfaz personalizada de alta fidelidad bajo una estética minimalista de alto contraste en negro y oro. La potencia transaccional es impulsada por WooCommerce, integrando funciones avanzadas como soporte multimoneda y una gestión de inventario dinámica. Gracias a esta combinación de tecnologías líderes, ING-SYSTEM ofrece un entorno de compra seguro, responsivo y sofisticado, diseñado para el consumidor contemporáneo que busca la excelencia en cada detalle. (2026)
